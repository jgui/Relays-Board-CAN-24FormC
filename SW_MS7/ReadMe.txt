
Content of this folder:

- Folder ./RelayCAN: "Microchip Studio" solution to generate the code for the ATmega88A on the relay board.
	Microchip Studio can be downloaded here:
	https://www.microchip.com/en-us/development-tools-tools-and-software/microchip-studio-for-avr-and-sam-devices

- Folder ./avr-can-lib: submodule used by the main project.

- Folder ./avr-liberty: submodule used by the main project.
